# Style Catalogue

Use this to pick a style for the user, based on the **vibe** and **formality** they want. Any style
can carry almost any content, so choose by feel, not by occasion. If the user named a style or a
clear preference, honour it. Otherwise infer from the tone they want and propose one. After
generating, always tell the user they can switch to any other style.

**Formality key:** **Low** = playful / casual, **Medium** = confident / modern, **High** =
restrained / refined / institutional.

## How to choose

1. **Formality first.** How serious should the board feel? A formal brief leans High (Reading Room,
   Linen Cut, Cobalt Glaze, Jade Lens); a playful or internal board leans Low (BlockFrame, Confetti
   Wedge, Crayon Stack, Mint Brut); most general work sits comfortably at Medium.
2. **Vibe second.** Narrow by the user's mood or colour preference (the Vibe column).
3. **When unsure, default to a versatile Medium style.** Riso Brut is the safe, designed default.
4. Always offer to switch afterwards. Swapping style means re rendering the same content in a
   different template's palette.
